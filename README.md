# AI for Legal Case Outcome Prediction ⚖️

This project predicts the outcome of legal cases using Machine Learning.

## 🚀 How to Run
1. Clone the repo:
   `git clone https://github.com/Nikhil270703/AI-For-legal-case-outcome-prediction.git`
2. Install dependencies:
   `pip install -r requirements.txt`
3. The models are hosted on Hugging Face and will be downloaded automatically when you run the inference script.
